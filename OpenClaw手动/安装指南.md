## Windows 安装（华为/联想/戴尔/小米等）

进入 `Windows/` 文件夹，按顺序运行：

| 步骤 | 文件 | 操作 |
|------|------|------|
| 1 | `1-安装Node.bat` | **右键→以管理员身份运行** |
| 2 | `2-安装Git.bat` | 双击运行 |
| 3 | `3-安装OpenClaw.bat` | 双击运行 |

每步完成后关掉窗口再运行下一步。

---

## macOS 安装（苹果电脑）

进入 `macOS/` 文件夹，双击 `install.command`。

首次运行如果提示"无法验证开发者"，右键→打开。

脚本自动检测 M芯片/Intel 并安装 Node + Git + OpenClaw。

## 安装后配置（两个平台通用）

### 配置大模型（智谱 GLM）

打开终端执行：

```bash
# 设置模型
openclaw config set model glm-5.1

# 设置 API Key（替换为你的 Key）
openclaw config set apiKey 你的API_Key

# 设置接口地址（注意：用 /coding/ 路径，这是 OpenClaw 编程专用接口）
openclaw config set baseUrl https://open.bigmodel.cn/api/coding/paas/v4

# 验证
openclaw config list
```

API Key 获取：https://open.bigmodel.cn/ → 注册 → API Keys

**可用模型**：
| 模型 | 说明 |
|------|------|
| glm-5.1 | 最强（推荐） |
| glm-5v-turbo | 快速，支持图片 |
| glm-4-flash | 免费 |

**重要**：智谱 provider 代码已从 `zhipu` 改为 `zai`，安装器内部已自动处理迁移。如果手动配置，请使用 `zai` 作为 provider 名。

### 配置飞书

```bash
# 方式一：扫码（推荐）
openclaw channels add feishu

# 方式二：手动编辑配置文件
```

手动配置 — 编辑 `~/.openclaw/openclaw.json`：
```json
{
  "channels": {
    "feishu": {
      "enabled": true,
      "appId": "cli_xxxxxxxxxxxxx",
      "appSecret": "你的App_Secret",
      "dmPolicy": "open"
    }
  }
}
```

飞书开放平台创建应用：https://open.feishu.cn/app
需要的权限：消息收发、通讯录（可选）

### 启动 Gateway

```bash
# 前台运行（调试）
openclaw gateway run

# 后台运行
openclaw gateway run --force

# 开机自启
openclaw gateway install
```

地址：http://127.0.0.1:18789

## 如果命令不生效（手动改配置文件）

配置文件位置：
- Windows：`C:\Users\你的用户名\.openclaw\`
- macOS：`~/.openclaw/`

**openclaw.json**：
```json
{
  "agents": {
    "defaults": {
      "model": {
        "primary": "zai/glm-5.1"
      }
    }
  }
}
```

**agents/main/agent/auth-profiles.json**：
```json
{
  "version": 1,
  "profiles": {
    "zai:manual": {
      "type": "api_key",
      "provider": "zai",
      "key": "你的API_Key"
    }
  }
}
```

**.env**（备选）：
```
ZAI_API_KEY=你的API_Key
```

## 常见问题

| 问题 | 解法 |
|------|------|
| Node.js 安装被拦截 | 关杀毒，右键→管理员运行 |
| npm install 失败 | 在线装：`npm install -g openclaw@2026.4.23 --registry=https://registry.npmmirror.com` |
| PATH 找不到 openclaw | 关终端重开，或重启电脑 |
| Gateway 启动失败 | 端口占用：`netstat -ano \| findstr 18789`（Win）/ `lsof -i :18789`（Mac） |
| macOS 提示无法验证 | 右键→打开，或 `xattr -d com.apple.quarantine 文件名` |
| 飞书消息收不到 | 检查事件订阅地址、权限、Gateway 状态 |
| acpx probe failed | 忽略，这是内置 ACP 插件探测失败，不影响正常使用 |
| 模型是 openai/gpt-5.4 | 之前装过 AutoClaw 的残留，用安装器重新配置国内模型即可 |
| gateway 显示 ready 但不响应 | 检查模型配置是否正确（provider/apiKey/baseUrl），openai 需要翻墙 |

## 安装器升级说明

已安装旧版（2026.4.15）的用户，直接运行新版安装器即可：
- 自动检测旧版本 → 自动升级到 2026.4.23
- 用户配置（API Key、飞书、模型）全部保留
- 记忆文件、会话历史不受影响（npm 全局包和 ~/.openclaw/ 目录分离）
